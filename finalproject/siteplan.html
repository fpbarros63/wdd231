Retro Game Library - Site Plan

Francisco de Paula Barros | WDD 231

1. Site Name and Purpose

Site Name: Retro Game Library

Reason: This name effectively conveys the purpose of the website, which is to serve as an interactive archive and personal collection of classic video games.

Site Purpose:

This website is designed to be a dynamic platform for retro gaming enthusiasts to learn about classic console games, view their metadata (such as platform, genre, and release year), and maintain their own personalized "Favorites" list via browser local storage.

2. Scenarios

Target audience questions that drive the content layout:

  • "How can I quickly find all the classic RPGs available in the library?"
  • "What year did Chrono Trigger come out and what console was it on?"
  • "How do I save a list of games I want to play later so I don't forget?"

3. Color Scheme

A palette balancing a dark blue aesthetic (evoking classic tech/consoles) with vibrant retro accents.

Primary Color
#2c3e50

Used for: Header backgrounds, primary text headings, and main footer.

Secondary Color
#e74c3c

Used for: Call-to-action buttons, active navigation links, and removed-favorite buttons.

Accent Color
#f39c12

Used for: Highlighted buttons (Add to Favs) and hover accents.

Background
#f4f6f7

Used for: Main body background, behind white game cards.

4. Typography

Two distinct Google fonts establish the retro yet readable theme:

Font: 'Press Start 2P'

Usage: Main site titles (H1) and the primary Branding Logo text to establish the retro gaming theme.

The Quick Brown Fox

Font: 'Open Sans'

Usage: All general body text (paragraphs, info lists, game descriptions) to ensure high readability and accessibility.

The quick brown fox jumps over the lazy dog.
